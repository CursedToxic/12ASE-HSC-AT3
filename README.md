# Locked In(c).
A basic study app designed to reduce distractions, improve organisation and promote focus.

## Installation
Clone the repository and navigate to the project directory:
12ASE-HSC-AT3/main.py

Then run the file after installing the dependencies in 'requirements.txt'.

Note: I'm on MacOS so there is no .exe file

## Features
- Title Screen
- Registration Screen
- Login Screen
- Clock Screen, which also doubles as a welcome screen after the user has logged in/registered
- Settings Page
- Dark Mode/Light Mode Toggle
- To-do List (can also be called a Task Tracker)
- Notes
- Flashcards
- Calendar with Event Planner

## Requirements
To run this program, you need to install the following dependencies:

- `customtkinter` for the main window.
- `random` for random enemy spawnpoints

### Install dependencies
To install the required dependencies, you can run:

```bash
pip install -r requirements.txt
``` 

Or install them manually (you might not even have to install random).
